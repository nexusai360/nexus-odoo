## Decisões de estrutura (fechadas, conforme feedback)

1. **Não existe página `/agente`.** O chat do agente é **a bubble**, e só ela.
   A tela cheia de conversas (`agente/page.tsx` + `client.tsx` + `conversation-list.tsx`)
   é **removida**.
2. **"Agente" no sidebar vira um grupo expansível** (igual "Agente Nex" do
   insights), com submenus: **Configuração · Chaves de API · Prompt · Consumo ·
   Playground**.
3. **RBAC:** o grupo "Agente" e todos os 5 submenus → **`super_admin` only**.
   A **bubble do agente** → **`super_admin` + `admin`**. Nenhum outro papel vê
   nem o grupo nem a bubble.
4. **Configuração deixa de ser página única** — vira 4 telas separadas
   (Configuração / Chaves de API / Prompt / Consumo) + Playground, navegáveis
   pelo submenu, como no nexus-insights.
5. **Selects:** portar `CustomSelect` e `SearchableSelect` do insights; **todo
   `<select>` nativo é eliminado**.
