# F5 — Rework de UI rodada 6 — Plano de Implementação (v3)

> **For agentic workers:** plano executado por **subagentes Opus 4.7** (todos
> em Opus — execução e review), despachados **em paralelo** após a fundação.
> Steps usam checkbox (`- [ ]`). SUB-SKILL: `superpowers:subagent-driven-development`.

**Goal:** Entregar todo o feedback pendente de UI da F5 — telas do Agente Nex
(Prompt, Recursos, KB, Consumo, Playground, Bubble) e o menu Integrações
inteiro (Canais/WhatsApp, Webhooks, MCP, API).

**Architecture:** Uma fase de **Fundação** (componentes compartilhados +
migrations + script de limpeza) executada sequencialmente; depois **blocos de
feature independentes** executados em paralelo por subagentes Opus 4.7. Clone
visual do `nexus-insights` onde há tela equivalente; reuso estrito do design
system; `ui-ux-pro-max` em cada componente.

**Tech Stack:** Next.js 16 (App Router), React, Tailwind v4, base-ui, Prisma
v7, framer-motion. SPEC: `docs/superpowers/specs/2026-05-19-f5-ui-rework-r6-spec.md`.
Referência visual: `/Users/joaovitorzanini/Developer/Claude Code/Nexus AI/Projetos Internos/nexus-insights`.

**Regras:** `ui-ux-pro-max` obrigatório em tudo de frontend. "Agente Nex" em
toda menção. Commits atômicos `feat(f5-ui):`/`fix(f5-ui):` por task.
`npx tsc --noEmit` + `npm run build` verdes ao fim de cada bloco. Cada
subagente roda `npx eslint` nos arquivos que tocou.
